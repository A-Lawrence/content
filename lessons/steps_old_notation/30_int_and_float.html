Floating-point numbers

When we read an integer value, we read a line with input() and then cast a string to integer using int(). When we read a floating-point number, we need to cast the string to float using float(): {% program %} x = float(input()) print(x) {% inputdata %} 1.492 {% endinputdata %} {% endprogram %}

You can write floats with very big or very small absolute value using the scientific notation. Eg., the distance from the Earth to the Sun is 1.496·1011, or 1.496e11 in Python. The mass of one molecule of the water is 2.99·10-23, or 2.99e-23 in Python.

You can also cast float objects to int objects. Of course it is achieved using the int() function: by discarding the fraction part using the . {% program %} print(int(1.3)) # gives 1 print(int(1.7)) # gives 1 print(int(-1.3)) # gives -1 print(int(-1.7)) # gives -1 {% endprogram %} The int() function simply discards the fractional part, as you see. It demonstrates so-called rounding towards zero behavior. {% Step 5 of X %}

Round and round

There's also a function round() that performs the usual rounding: {% program %} print(round(1.3)) # gives 1 print(round(1.7)) # gives 2 print(round(-1.3)) # gives -1 print(round(-1.7)) # gives -2 {% endprogram %} {% Step 5 of X %}

Floats can confuse you

Floating-point real numbers can't be represented with exact precision due to hardware limitations. This can lead to cumbersome effects. See the official Python docs for the details. {% program %} print(0.1 + 0.2) # gives 0.30000000000000004 {% endprogram %} {% Step 6 of X %}

Math module

Python has a ton of functions for calculations with floats. They can be found in the math module.

To use this module, we need to import it first by writing the following instruction at the beginning of the program: 

import math

For example, if we want to find a ceiling value for x — the smallest integer not less than x — we call the appropriate function from the math module: math.ceil(x). {% program %} import math x = math.ceil(4.2) print(x) print(math.ceil(1 + 3.8)) {% endprogram %} {% instructions %} 1. More ways to import math

There's another way to use functions from modules: to import the certain functions by naming them {% program %} from math import ceil x = 7 / 2 y = ceil(x) print(y) {% endprogram %}
