The loop called for

In the previous lessons we dealt with sequential programs and conditions. Quite often programs need to repeat some block several times. That's where the loops come in handy. To sum first n numbers, you of course can write something like {% program %} sum = 0 sum = sum + 1 sum = sum + 2 sum = sum + 3 sum = sum + 4 sum = sum + 5 print(sum) {% endprogram %} But it is inconvenient. Imagine you had to sum the first 100 numbers, or iterate somehow. You surely wouldn't write the 100 lines! {% Step 2 of X %}

The loop called for

If you have to do fixed amount of operations — say, sum the 100 first numbers — the for loop saves the day. There are for and while loop operators in Python, in this lesson we cover for.

for loop iterates over any sequence. Let's see the simplest example of using for: {% program %} i = 1 for color in 'red', 'orange', 'yellow', 'green', 'cyan', 'blue', 'violet': print('#', i, 'color of rainbow is', color) i = i + 1 {% endprogram %} {% instructions %} Press "Run" to see what will output such a program! {% endinstructions %} {% Step 3 of X %}

The loop called for

{% program %} i = 1 for color in 'red', 'orange', 'yellow', 'green', 'cyan', 'blue', 'violet': print('#', i, 'color of rainbow is', color) i = i + 1 {% endprogram %}

In this example the variable color iterates over the values 'red', 'orange' etc, and for each value the underlying loop block is executed. Loop block contains two operators: call of the function print() that prints new data and assignment operator that increases the value of the variable i by one. {% Step 4 of X %}

Iterate over sequences

for loop iterates over any sequence. For instance, any string in Python is a sequence of its characters, so we can iterate over them using for: {% program %} for character in 'hello': print(character) {% endprogram %} {% instructions %} {% endinstructions %} {% Step 6 of X %}

Range-1

{% program %} for i in range(5, 8): print(i, i ** 2) print('end of loop') {% endprogram %}

Another use case for a for-loop is to iterate some integer variable in increasing or decreasing order. Such a sequence of integer can be created using the function range(min_value, max_value): {% Step 7 of X %}

Range-2

{% program %} for i in range(4,8): print(i) # This will print 4, 5, 6 and 7 {% endprogram %}

Function range(min_value, max_value) generates a sequence with numbers min_value, min_value + 1, ..., max_value - 1. The last number is not included. {% Step 8 of X %}

Range-3

{% program %} for i in range(3): print(i) # And this will print 0, 1 and 2 {% endprogram %}

Don't create empty ranges

{% program %} for i in range(-5): print('Hello, world!') {% endprogram %}

If you pass an empty sequence to range(), like range(-5) or range(7, 3), the for-block won't be executed. Generally, you're not interested in empty range. {% instructions %} {% endinstructions %} {% Step 10 of X %}

When *= doesn't mean a kiss

{% program %} result = 0 debt = 1000 prod = 1 divine = 100 n = 5 for i in range(1, n + 1): result += i # the same as result = result + i debt -= i # the same as debt = debt - i prod *= i # the same as prod = prod * i divine /= i # the same as divine = divine / i print(result) print(debt) print(prod) print(divine) {% endprogram %} In the previous program, we've written result += i instead of result = result + i. The operators +=, -=, *=, /= are common shortcuts; see the code section. {% Step 12 of X %}

Step can be negative

To iterate over a decreasing sequence, we can use an extended form of range() with three arguments — range(start_value, end_value, step). When omitted, the step is implicitly equal to 1. However, it can be any non-zero value. The loop always includes start_value and excludes end_value during iteration: {% program %} for i in range(10, 0, -2): print(i) # 10 # 8 # 6 # 4 # 2 {% endprogram %} {% instructions %} Click "Run" to see what happens in output! {% endinstructions %} {% Step 13 of X %}

Setting the function print()

By default, the print() function prints all its arguments separating them by a space and then puts the newline symbol after it.
